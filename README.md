# My Portfolio

Welcome! This repository hosts my personal portfolio website, showcasing my experience building cloud-native architectures, automated DevOps pipelines, and data processing solutions.

## 🛠️ Tech Stack & Architecture
* **Cloud & Serverless:** AWS (Lambda, ECS Fargate, Step Functions, DynamoDB, S3, RDS, SQS/SNS, API Gateway)
* **Infrastructure as Code:** AWS CDK, CloudFormation
* **Data & Analytics:** Azure Synapse, Azure ADLS Gen2
* **CI/CD & DevOps:** GitHub Actions, AWS CodeBuild

## 🚀 Features
* Responsive UI displaying core technical competencies and project highlights.
* Dynamic, styled badges for technology tracking.
* Clean, semantic HTML and CSS structure.
